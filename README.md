# 🌟 Ruixen UI

Beautifully crafted, production-ready UI components to elevate your web projects. Built with **Next.js**, **Tailwind CSS**, **TypeScript**, and **Framer Motion** — fully customizable and open-source.

![Ruixen UI Screenshot](https://i.ibb.co/HthN5FV/Screenshot-2024-09-01-223237.png)

---

## ✨ Features

- 🚀 Modern, responsive components
- 🎨 Built with Tailwind CSS for easy theming
- 💡 Smooth animations using Framer Motion
- 🔧 Type-safe components with TypeScript
- ⚡ Ready-to-use examples
- 🛠️ Fully customizable & extendable
- 🌍 Open-source under MIT License

---

npm install @ruixen/ui
