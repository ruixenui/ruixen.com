[MCP](https://modelcontextprotocol.com/) is an open protocol that standardizes how applications provide context to LLMs.

This is useful for Ruixen UI because you can now give your AI-assisted IDE direct access to all Ruixen UI components so that it can generate code with minimal errors.

<Step>Add the following to your MCP config file:</Step>

```json
{
  "mcpServers": {
    "@ruixenui/mcp": {
      "command": "npx",
      "args": ["-y", "@ruixenui/mcp@latest"]
    }
  }
}
```

<Step>Restart your IDE</Step>

</Steps>
