
## Props

| Prop        | Type                                                  | Default         | Description               |
| ----------- | ----------------------------------------------------- | --------------- | ------------------------- |
| `logo`      | `React.ReactNode`                                     | Default logo    | Custom logo element       |
| `links`     | `{ label: string; href: string; active?: boolean }[]` | Default links   | Navigation links array    |
| `status`    | `"operational" \| "degraded" \| "down"`               | "operational"   | System status indicator   |
| `metrics`   | `{ uptime: string; latency: string }`                 | Default metrics | System metrics to display |
| `className` | `string`                                              | -               | Additional CSS classes    |

## Features

- Status indicator with color coding
- System metrics display (uptime, latency)
- Clean, professional design
- Real-time status updates
- Light and dark mode support
