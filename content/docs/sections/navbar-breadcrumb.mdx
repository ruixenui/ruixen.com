## Usage

```tsx
import NavbarBreadcrumb from "@/components/ruixen/navbar-breadcrumb";

export default function MyNavbar() {
  return <NavbarBreadcrumb />;
}
```

## Props

| Prop          | Type                                | Default         | Description                 |
| ------------- | ----------------------------------- | --------------- | --------------------------- |
| `logo`        | `React.ReactNode`                   | Default logo    | Custom logo element         |
| `breadcrumbs` | `{ label: string; href: string }[]` | Default crumbs  | Breadcrumb navigation items |
| `actions`     | `React.ReactNode`                   | Default buttons | Custom action buttons       |
| `className`   | `string`                            | -               | Additional CSS classes      |

## Features

- Breadcrumb-style navigation
- Chevron separators
- Hierarchical structure display
- Clean, minimal design
- Light and dark mode support
