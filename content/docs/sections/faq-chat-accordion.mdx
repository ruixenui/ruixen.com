## Usage

```tsx
import FaqChatAccordion from "@/components/ruixen/faq-chat-accordion";

export default function MyFAQ() {
  return <FaqChatAccordion className="max-w-[700px]" />;
}
```

## Custom Data

```tsx
const myFaqs = [
  {
    id: 1,
    question: "How do I get started?",
    answer: "Sign up for a free account and follow the onboarding guide.",
    icon: "üöÄ",
    iconPosition: "right",

  {
    id: 2,
    question: "Is there a free plan?",
    answer: "Yes, we offer a generous free tier for individual developers.",
  },
];

<FaqChatAccordion data={myFaqs} timestamp="Updated daily" />;
```

## Props

| Prop                | Type        | Default                | Description                              |
| ------------------- | ----------- | ---------------------- | ---------------------------------------- |
| `data`              | `FAQItem[]` | Built-in sample data   | Array of FAQ items to display            |
| `className`         | `string`    | -                      | Additional CSS classes for the container |
| `timestamp`         | `string`    | `"Every day, 9:01 AM"` | Timestamp text shown above the accordion |
| `questionClassName` | `string`    | -                      | Custom classes for question pills        |
| `answerClassName`   | `string`    | -                      | Custom classes for answer bubbles        |

## FAQ Item Structure

## Features

- **Chat Bubble Answers**: Messenger-style rounded bubbles for answer text
- **Emoji Stickers**: Optional emoji decorations on question pills
- **Plus/Minus Toggle**: Animated icon toggle via CSS data-attributes
- **No JS Animation Library**: Uses native AccordionContent animation
- **Customizable Styles**: Override question and answer styles via className props
