## Props

| Prop          | Type       | Default         | Description                    |
| ------------- | ---------- | --------------- | ------------------------------ |
| `roomName`    | `string`   | `"Living Room"` | Name of the room               |
| `temperature` | `number`   | `22.5`          | Current temperature in Celsius |
| `devices`     | `Device[]` | -               | Array of smart devices         |
| `scenes`      | `Scene[]`  | -               | Array of scene configurations  |
