## Props

| Prop       | Type                         | Default | Description                       |
| ---------- | ---------------------------- | ------- | --------------------------------- |
| `events`   | `WallEvent[]`                | `[]`    | Initial events to display         |
| `onAdd`    | `(event: WallEvent) => void` | -       | Callback when an event is created |
| `onRemove` | `(id: string) => void`       | -       | Callback when an event is deleted |
| `sound`    | `boolean`                    | `true`  | Enable soft tick on interactions  |

### WallEvent

| Property | Type     | Description                 |
| -------- | -------- | --------------------------- |
| `id`     | `string` | Unique identifier           |
| `title`  | `string` | Event title                 |
| `date`   | `string` | Date in "YYYY-MM-DD" format |
