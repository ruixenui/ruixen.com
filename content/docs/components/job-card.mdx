## Props

| Prop         | Type     | Default               | Description         |
| ------------ | -------- | --------------------- | ------------------- |
| `title`      | `string` | `"Software Engineer"` | Job title           |
| `company`    | `string` | `"Tech Corp"`         | Company name        |
| `rate`       | `string` | `"$80k - $100k"`      | Salary range        |
| `location`   | `string` | `"Remote"`            | Job location        |
| `type`       | `string` | `"Full-time"`         | Employment type     |
| `experience` | `string` | `"2+ years"`          | Required experience |
| `logoUrl`    | `string` | `""`                  | Company logo URL    |

## Features

- **Company Avatar**: Displays company logo or fallback
- **Job Details**: Title, company, salary, location, type, experience
- **Action Buttons**: Save, Share, and Apply buttons
- **Responsive Design**: Works on all screen sizes
- **Clean Layout**: Professional and organized appearance
