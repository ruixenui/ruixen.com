## Props

| Prop       | Type                       | Default | Description                                  |
| ---------- | -------------------------- | ------- | -------------------------------------------- |
| `value`    | `string`                   | -       | Selected date as "YYYY-MM-DD"                |
| `onChange`  | `(date: string) => void`  | -       | Callback fired when a day is selected        |
| `sound`    | `boolean`                  | `true`  | Enable soft tick on selection and navigation |
