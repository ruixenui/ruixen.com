## Props

| Prop            | Type                      | Default | Description                                          |
| --------------- | ------------------------- | ------- | ---------------------------------------------------- |
| `min`           | `number`                  | `-90`   | Minimum value                                        |
| `max`           | `number`                  | `90`    | Maximum value                                        |
| `step`          | `number`                  | `1`     | Step increment                                       |
| `value`         | `number`                  | -       | Controlled value (use with `onChange`)               |
| `defaultValue`  | `number`                  | `0`     | Initial value for uncontrolled mode                  |
| `suffix`        | `string`                  | `"°"`   | Unit suffix displayed after the value                |
| `sound`         | `boolean`                 | `true`  | Enable mechanical tick sounds on scrub               |
| `labelInterval` | `number`                  | `10`    | Interval for labeled ticks (largest, with numbers)   |
| `majorInterval` | `number`                  | `5`     | Interval for major ticks (medium height, no numbers) |
| `onChange`      | `(value: number) => void` | -       | Callback fired when the value changes                |
