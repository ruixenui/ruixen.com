## Props

| Prop          | Type             | Default | Description                                 |
| ------------- | ---------------- | ------- | ------------------------------------------- |
| `menuItems`   | `MenuItemType[]` | `[]`    | Array of menu items with nested structure   |
| `collapsed`   | `boolean`        | `false` | Whether the menu is collapsed               |
| `onItemClick` | `function`       | -       | Callback function when menu item is clicked |
