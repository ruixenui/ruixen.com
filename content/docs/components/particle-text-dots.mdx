## Props

| Prop        | Type                | Default  | Description                     |
| ----------- | ------------------- | -------- | ------------------------------- |
| `text`      | `string`            | `"3"`    | The text to render as particles |
| `variant`   | `"dark" \| "light"` | `"dark"` | Color theme for the particles   |
| `className` | `string`            | -        | Additional CSS classes          |
