## Props

| Prop       | Type                        | Default            | Description                       |
| ---------- | --------------------------- | ------------------ | --------------------------------- |
| `data`     | `MagicNode[]`               | Default magic data | Tree data with sparkle properties |
| `onSelect` | `(node: MagicNode) => void` | -                  | Callback when node is selected    |

## Features

- **Sparkle Effects**: Special nodes with magical sparkle indicators
- **Interactive Animations**: Hover and tap animations
- **Star Selection**: Selected items show with star icon
- **Smooth Transitions**: Framer Motion animations
- **Color-coded Borders**: Purple accent for magical elements
- **Default Data**: Comes with sample magical folder structure
