## Props

| Prop           | Type                   | Default | Description                                 |
| -------------- | ---------------------- | ------- | ------------------------------------------- |
| `value`        | `Date`                 | -       | Controlled date value (use with `onChange`) |
| `defaultValue` | `Date`                 | `now`   | Initial date for uncontrolled mode          |
| `sound`        | `boolean`              | `true`  | Enable soft mechanical tick on each step    |
| `minYear`      | `number`               | `2015`  | Earliest year available for scrubbing       |
| `maxYear`      | `number`               | `2035`  | Latest year available for scrubbing         |
| `onChange`     | `(date: Date) => void` | -       | Callback fired when any segment changes     |
