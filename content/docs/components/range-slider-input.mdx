## Props

| Prop       | Type                                          | Default          | Description                    |
| ---------- | --------------------------------------------- | ---------------- | ------------------------------ |
| `label`    | `string`                                      | `"Select Range"` | The label for the range slider |
| `min`      | `number`                                      | `0`              | Minimum value for the range    |
| `max`      | `number`                                      | `100`            | Maximum value for the range    |
| `step`     | `number`                                      | `1`              | Step increment for the slider  |
| `unit`     | `string`                                      | `undefined`      | Unit to display with values    |
| `value`    | `{ from: number; to: number }`                | `undefined`      | Controlled value for the range |
| `onChange` | `(val: { from: number; to: number }) => void` | `undefined`      | Callback when range changes    |
