## Features

- **Complete Task Form**: Task name, assignee, category, due date, and priority
- **Date Picker Integration**: Calendar popover for due date selection
- **Select Components**: Dropdown menus for assignee, category, and priority
- **Icon Integration**: Lucide icons for visual enhancement
- **Responsive Design**: Works well on different screen sizes
- **Sheet Component**: Slide-out panel for better UX

## Form Fields

- **Task Name**: Text input for task description
- **Assignee**: Select dropdown with user options
- **Category**: Select dropdown for task categorization
- **Due Date**: Calendar picker with date selection
- **Priority**: Select dropdown with priority levels (Low, Medium, High)

## Use Cases

Perfect for:

- **Project Management**: Task creation and assignment
- **Team Collaboration**: Assigning tasks to team members
- **Workflow Management**: Organizing tasks by category and priority
- **Deadline Tracking**: Setting and managing due dates
- **Task Organization**: Structured task creation process
