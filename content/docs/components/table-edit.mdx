## Features

- **Inline Editing**: Edit table cells directly without dialogs
- **Checkbox Selection**: Select multiple rows for batch operations
- **Dropdown Actions**: Edit or delete rows via context menu
- **Status Management**: Update user status with dropdown select
- **Real-time Updates**: Changes reflect immediately in the table

## Props

| Prop          | Type     | Default        | Description                      |
| ------------- | -------- | -------------- | -------------------------------- |
