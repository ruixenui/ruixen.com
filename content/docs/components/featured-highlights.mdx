## Props

| Prop       | Type            | Default                                           | Description                       |
| ---------- | --------------- | ------------------------------------------------- | --------------------------------- |
| `heading`  | `string`        | `"Powerful Features to Accelerate Your Workflow"` | Main heading text                 |
| `features` | `FeatureItem[]` | `defaultFeatures`                                 | Array of feature items to display |

## FeatureItem Interface

| Prop          | Type     | Description               |
| ------------- | -------- | ------------------------- |
| `id`          | `number` | Unique identifier         |
| `title`       | `string` | Feature title             |
| `image`       | `string` | Feature preview image URL |
| `description` | `string` | Feature description text  |
