## Features

- **File Upload**: Select images from your device
- **Interactive Cropping**: Drag and position crop area
- **Zoom Control**: Slider to zoom in/out on image
- **Aspect Ratios**: Preset ratios (1:1, 4:3, 16:9) or free form
- **Live Preview**: See cropped result before downloading
- **Download**: Save cropped image as PNG

## Usage

The component uses `react-easy-crop` for smooth cropping interactions. Users can:

1. Upload an image file
2. Adjust zoom with slider
3. Select aspect ratio
4. Crop the image
5. Download the result
