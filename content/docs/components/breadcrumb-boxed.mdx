## Usage

```tsx
import BreadcrumbBoxed from "@/components/ruixen/breadcrumb-boxed";

const items = [
  { label: "Home", href: "/" },
  { label: "Components", href: "/components" },
  { label: "Breadcrumb" },
];

export default function MyBreadcrumb() {
  return <BreadcrumbBoxed items={items} rounded="full" />;
}
```

## Props

| Prop        | Type                             | Default | Description               |
| ----------- | -------------------------------- | ------- | ------------------------- |
| `items`     | `BreadcrumbBoxedItem[]`          | -       | Array of breadcrumb items |
| `rounded`   | `"sm" \| "md" \| "lg" \| "full"` | `"lg"`  | Border radius style       |
| `className` | `string`                         | -       | Additional CSS classes    |

### BreadcrumbBoxedItem

| Property | Type     | Description               |
| -------- | -------- | ------------------------- |
| `label`  | `string` | Display text for the item |
| `href`   | `string` | Optional link URL         |

## Features

- **Segment Relief**: Each item lifts from the muted container on hover — `bg-background` fades in with a subtle `shadow-sm`, creating physical depth
- **Underline Wipe**: A 2px accent line wipes in from the left edge using `cubic-bezier(0.65, 0, 0.35, 1)` — starts slow, bursts through the middle, decelerates. A "painting" feel
- **Active Indicator**: The current page (last item) shows a permanent underline, reinforcing "you are here" without hover
- **Vertical Dividers**: 1px line separators between segments — the most "segmented control" separator, distinct from chevrons and slashes in the other breadcrumbs
- **Matched Radii**: Inner segment radius scales with the container radius — `rounded-full` container gets `rounded-full` segments for cohesive pill shapes
- **Pure CSS**: No React state, no event handlers — all hover interactions via Tailwind `group-hover` and CSS transitions
- **Accessibility**: `aria-current="page"` on active item, `aria-hidden` dividers, semantic `nav` + `ol` structure
