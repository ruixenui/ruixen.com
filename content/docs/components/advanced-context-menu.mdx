## Props

| Prop           | Type         | Default | Description                                  |
| -------------- | ------------ | ------- | -------------------------------------------- |
| `items`        | `MenuItem[]` | `[]`    | Array of menu items                          |
| `trigger`      | `ReactNode`  | -       | Element that triggers the context menu       |
| `onItemSelect` | `function`   | -       | Callback function when menu item is selected |
