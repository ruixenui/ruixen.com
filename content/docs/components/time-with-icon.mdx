## Props

| Prop          | Type     | Default         | Description                        |
| ------------- | -------- | --------------- | ---------------------------------- |
| `label`       | `string` | `"Select Time"` | The label for the time input       |
| `defaultTime` | `string` | `undefined`     | Default time value in HH:MM format |

## Features

- **Dynamic Icons**: Automatically switches between sun (day) and moon (night) icons based on time
- **Time Range Detection**: Day hours (6 AM - 6 PM) show sun icon, night hours show moon icon
- **Default Clock Icon**: Shows clock icon when no time is selected
- **Native Time Picker**: Uses browser's native time input for best UX
