## Features

- **Drag & Drop**: Drag images directly into the upload area
- **Multi-Upload**: Upload multiple images at once
- **Image Preview**: Grid view of uploaded images
- **Crop Modal**: Crop individual images with modal interface
- **Delete Images**: Remove unwanted images
- **Cropped Preview**: Shows cropped version when available
- **Hover Actions**: Edit and delete buttons on hover

## Usage

This component combines file upload with image cropping:

1. Drag & drop or click to select images
2. Images appear in a grid preview
3. Hover over any image to crop or delete
4. Click crop icon to open cropping modal
5. Adjust crop area and save
6. Cropped version replaces original in preview
