## Props

| Prop   | Type       | Default | Description                 |
| ------ | ---------- | ------- | --------------------------- |
| `node` | `TreeNode` | -       | Tree node with tooltip data |

## Features

- **Tooltip Support**: Each node can have custom tooltip text
- **Expandable Structure**: Folders can be expanded/collapsed
- **Smooth Animations**: Framer Motion powered transitions
- **Hover Effects**: Visual feedback on hover
- **File/Folder Icons**: Clear visual distinction
- **Nested Structure**: Supports deeply nested hierarchies
- **Demo Data**: Includes sample tree structure
