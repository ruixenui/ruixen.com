## Props

| Prop          | Type     | Default                                       | Description                                           |
| ------------- | -------- | --------------------------------------------- | ----------------------------------------------------- |
| `title`       | `string` | `"Experience the Spectrum"`                   | The main heading text                                 |
| `description` | `string` | `"Discover a full range of possibilities..."` | The description text                                  |
| `buttons`     | `array`  | `[...]`                                       | Array of button objects with label, href, and variant |
| `colorScheme` | `string` | `"rainbow"`                                   | Color scheme for the spectrum effect                  |
