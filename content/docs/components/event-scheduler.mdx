## Features

- **Two-Panel Layout**: Event creation and event list side-by-side
- **Date & Time Selection**: Full calendar and time picker
- **Event Management**: Add and delete events
- **Event List**: View all scheduled events with timestamps
- **Form Validation**: Prevents empty events
- **Auto-Reset**: Form clears after successful creation

## Props

| Prop            | Type                     | Default | Description                 |
| --------------- | ------------------------ | ------- | --------------------------- |
| `defaultEvents` | `Event[]`                | `[]`    | Initial events              |
| `defaultTitle`  | `string`                 | `""`    | Default event title         |
| `defaultDate`   | `Date`                   | -       | Default date selection      |
| `defaultHour`   | `string`                 | `"12"`  | Default hour                |
| `defaultMinute` | `string`                 | `"00"`  | Default minute              |
| `defaultAMPM`   | `"AM" \| "PM"`           | `"AM"`  | Default AM/PM               |
| `onAddEvent`    | `(event: Event) => void` | -       | Callback when event added   |
| `onDeleteEvent` | `(id: number) => void`   | -       | Callback when event deleted |
| `disabled`      | `boolean`                | `false` | Disable all interactions    |
