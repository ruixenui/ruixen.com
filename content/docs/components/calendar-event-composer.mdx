## Props

| Prop             | Type                                | Default         | Description                                           |
| ---------------- | ----------------------------------- | --------------- | ----------------------------------------------------- |
| `date`           | `Date`                              | `new Date()`    | Day rendered in the inspector's date field            |
| `startHour`      | `number`                            | `10`            | First hour shown in the timeline (0–23)               |
| `endHour`        | `number`                            | `19`            | Last hour shown in the timeline (0–23)                |
| `initialEvents`  | `ComposerEvent[]`                   | `[]`            | Events pre-populated on first mount                   |
| `categories`     | `ComposerCategory[]`                | 5 macOS palette | Category options shown in the picker                  |
| `onEventsChange` | `(events: ComposerEvent[]) => void` | -               | Fires whenever events are created, edited, or resized |
| `className`      | `string`                            | -               | Extra classes merged onto the outer container         |

## Interactions

- **Click empty timeline** — creates a 60-minute event snapped to the nearest 15-minute mark and opens the inspector focused on the title input.
- **Click an existing event** — re-opens the inspector with that event's data.
- **Drag an event** — moves it within the day, snapping to 15-minute intervals.
- **Drag the bottom edge** — resizes the event's end time (minimum 30 minutes).
- **Escape** — closes the inspector without deselecting the block.
