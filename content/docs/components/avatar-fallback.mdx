## Usage

```tsx
import AvatarFallbackOnly from "@/components/ruixen/avatar-fallback";

export default function MyAvatar() {
  return <AvatarFallbackOnly fallback="KK" />;
}
```

## Props

| Prop        | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `fallback`  | `string` | `"KK"`  | Fallback text initials |
| `className` | `string` | -       | Additional CSS classes |
