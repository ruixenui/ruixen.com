## Props

| Prop       | Type     | Default                              | Description                |
| ---------- | -------- | ------------------------------------ | -------------------------- |
| `title`    | `string` | `"Build Stunning Interfaces"`        | Title of the showcase item |
| `subtitle` | `string` | `"Harness the power of elegant..."`  | Subtitle or description    |
| `image`    | `string` | `"/ruixenui-bg.png"`                 | Image URL for the showcase |
| `badge`    | `object` | `{ text: "New", variant: "orange" }` | Badge configuration        |
| `href`     | `string` | `"#"`                                | Link URL                   |
| `id`       | `string` | -                                    | Unique identifier          |
