## Usage

The Clean Tag Input component provides an elegant way to input multiple tags or keywords. Users can type text and press Enter to add a tag, or press Backspace when the input is empty to remove the last tag.

```tsx
<CleanTagInput />
```

## Features

- Add tags by pressing Enter
- Remove tags by clicking the X button or pressing Backspace when input is empty
- Clean, minimal design with subtle hover effects
- Dark mode support
- Accessible tag removal
- Flexible layout that adapts to container width
