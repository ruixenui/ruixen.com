## Props

| Prop       | Type                       | Default | Description                                  |
| ---------- | -------------------------- | ------- | -------------------------------------------- |
| `onSubmit` | `(card: CardData) => void` | -       | Callback with card data on successful submit |
| `amount`   | `string`                   | `$0.00` | Amount displayed on the pay button           |
| `sound`    | `boolean`                  | `true`  | Enable soft tick on keystrokes               |
