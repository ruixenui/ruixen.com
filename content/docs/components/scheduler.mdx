## Features

- **Popover Interface**: Compact popup for scheduling
- **24-Hour Time**: Hour and minute selection
- **Edit Mode**: Modify existing events
- **Event Cards**: Display scheduled events in cards
- **CRUD Operations**: Create, read, update, and delete events

## Props

| Prop            | Type                     | Default | Description                 |
| --------------- | ------------------------ | ------- | --------------------------- |
| `defaultEvents` | `Event[]`                | `[]`    | Initial events              |
| `defaultHour`   | `string`                 | `"12"`  | Default hour                |
| `defaultMinute` | `string`                 | `"00"`  | Default minute              |
| `onAddEvent`    | `(event: Event) => void` | -       | Callback when event added   |
| `onEditEvent`   | `(event: Event) => void` | -       | Callback when event edited  |
| `onDeleteEvent` | `(id: number) => void`   | -       | Callback when event deleted |
