## Usage

The Better Time Picker component provides a more intuitive way to select time with separate dropdowns for hours and minutes. It features a clean design with 5-minute increments for the minutes dropdown to make selection faster.

```tsx
<BetterTimePicker />
```

## Features

- Separate hour and minute dropdowns
- 24-hour format for hours
- 5-minute increments for minutes
- Clock icon for better visual indication
- Dark mode support
- Accessible dropdown menus
