## Features

- **Dialog Details**: View full user details in a modal dialog
- **Fixed Header/Footer**: Header and footer stay visible while scrolling
- **Status Badges**: Color-coded status indicators (Active/Inactive/Suspended)
- **Checkbox Selection**: Select rows for batch operations
- **Scrollable Body**: Handle large datasets with smooth scrolling

## Props

| Prop          | Type     | Default        | Description                      |
| ------------- | -------- | -------------- | -------------------------------- |
| `data`        | `User[]` | `defaultUsers` | Array of user data               |
| `title`       | `string` | `"User Table"` | Table header title               |
| `footerTotal` | `string` | `"$2,500.00"`  | Total amount displayed in footer |
