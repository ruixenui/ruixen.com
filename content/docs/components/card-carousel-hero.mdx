## Props

| Prop          | Type     | Default                                              | Description                                              |
| ------------- | -------- | ---------------------------------------------------- | -------------------------------------------------------- |

| `description` | `string` | `"Explore our collection of powerful components..."` | The description text                                     |
| `cards`       | `array`  | `[...]`                                              | Array of card objects with title, description, and image |
| `buttons`     | `array`  | `[...]`                                              | Array of button objects with label, href, and variant    |
