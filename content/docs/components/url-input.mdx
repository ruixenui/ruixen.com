## Features

- **Automatic Favicon Detection**: Displays the website's favicon when a valid URL is entered
- **Smart URL Parsing**: Handles URLs with or without protocol prefixes
- **Fallback Icon**: Shows a globe icon when no favicon is available
- **Error Handling**: Gracefully handles invalid URLs and missing favicons
