## Props

| Prop       | Type       | Default      | Description                                  |
| ---------- | ---------- | ------------ | -------------------------------------------- |
| `isOpen`   | `boolean`  | `false`      | Whether the drawer is open                   |
| `onClose`  | `function` | -            | Callback function when drawer is closed      |
| `onSubmit` | `function` | -            | Callback function when feedback is submitted |
| `title`    | `string`   | `"Feedback"` | Title of the feedback drawer                 |
