## Props

| Prop          | Type     | Default                                     | Description                                           |
| ------------- | -------- | ------------------------------------------- | ----------------------------------------------------- |
| `title`       | `string` | `"Video Showcase Experience"`               | The main heading text                                 |
| `description` | `string` | `"Immerse yourself in stunning visuals..."` | The description text                                  |
| `videoSrc`    | `string` | `""`                                        | Source URL for the background video                   |
| `buttons`     | `array`  | `[...]`                                     | Array of button objects with label, href, and variant |
