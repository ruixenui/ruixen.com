## Usage

The Inline Copy Input component provides a clean, user-friendly way to display text that users might want to copy, such as commands, URLs, or tokens. It features a built-in copy button that provides visual feedback when clicked.

```tsx
<InlineCopyInput />
```

## Features

- Read-only input field with predefined content
- Inline copy button with feedback
- Success state with color change
- Dark mode support
- Simple, clean design
- Accessible button for copying content
