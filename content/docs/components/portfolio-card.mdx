## Props

| Prop        | Type      | Default                           | Description                       |
| ----------- | --------- | --------------------------------- | --------------------------------- |
| `name`      | `string`  | `"Ruixen"`                        | Name of the person                |
| `role`      | `string`  | `"Software Developer Engineer"`   | Professional role or title        |
| `image`     | `string`  | -                                 | Profile image URL                 |
| `status`    | `string`  | `"Available for Remote Projects"` | Current availability status       |
| `skills`    | `Skill[]` | -                                 | Array of skills with levels (1-5) |
| `portfolio` | `string`  | -                                 | Portfolio or website URL          |
