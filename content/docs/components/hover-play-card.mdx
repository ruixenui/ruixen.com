## Usage

```tsx
import { HoverPlayCard } from "@/components/ruixen/hover-play-card";

export default function MyVideoCard() {
  return (
    <HoverPlayCard
      src="https://example.com/video.mp4"
      poster="https://example.com/thumbnail.jpg"
      loop
    />
  );
}
```

## Props

| Prop           | Type      | Default | Description                                 |
| -------------- | --------- | ------- | ------------------------------------------- |
| `src`          | `string`  | -       | **Required.** URL of the video file         |
| `poster`       | `string`  | -       | Optional thumbnail image URL                |
| `loop`         | `boolean` | `false` | Whether video should loop                   |
| `mutedOnHover` | `boolean` | `true`  | Whether to mute video during hover autoplay |
| `sound`        | `boolean` | `true`  | Enable audio tick feedback on interactions  |

## Features

- Hover autoplay — video plays muted when cursor enters, pauses on leave
- Click to unmute and play with sound, click again to pause
- Spring-animated center play/pause icon with glass circle
- Progress line at bottom edge — appears on hover/play, tracks position
- Inline SVG icons — no lucide dependency
- CSS custom property border that adapts to light/dark theme
- Audio tick on play/pause interactions
- Respects browser autoplay policies (muted autoplay)
